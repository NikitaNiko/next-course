Next.js

Установка:

npx init -y
npm i react react-dom next

Добавим в скрипт строку '"dev": "next dev"' и по команде "npm run dev" запускаем приложение

Папка pages - корневая папка приложения next, в ней находятся все страницы и путь к эти страницам === названию их файлов, т.е. index = /; users = /users и т.д
Папки внутри pages - это вложенные маршруты.
Можно создавать страницы, в ссылку которых добавляется аргумент. Название файла должно быть в [] скобках, но любое!

_app.js - обертка над всем приложением (см. документацию next). В нее импортируются глобальные стили для всех страниц

Чтобы подгрузить стиль в конкретный компонент, нужно, чтобы он назывался по шаблону: (имя стиля).modules.css (иначе он будет воспринят как глобальный). Далее применить к тегу как класс 
import stl from '../styles/A.module.css'
className={stl.link}

Можно создать свою страницу с несуществующей страницей. Для этого необходимо создать файл с названием 404.js в pages и внутри создать функцию, которая будет возвращать jsx объект со страницей, которую необходимо будет показывать

Для деплоя: 
    В файле package.json в скрипты добавляем еще 2 скрипта:
    "build": "next build",          - Финальная сборка приложения
    "start": "next start"           - Для запуска приложения    

    В git ignore должны быть следующие обязательные строчки:
    /node_modules
    /.env
    /.idea  
    /.next
